# PM Interview Review Lab Project Setup

Project name:

`PM Interview Review Lab`

Purpose:

Review completed PM interview sessions from the private tracker repo and produce practical improvement actions.

## Source files to use

Custom instructions source:

`docs/PM_INTERVIEW_REVIEW_LAB_PROJECT_INSTRUCTIONS.md`

Reusable review prompt:

`templates/session-tracker/review_lab_prompt.md`

Do not upload runtime scripts, archives, old project bundles, or the main PM Interview Helper upload bundle into this Review Lab Project.

## Recommended setup

1. Create a new ChatGPT Project named `PM Interview Review Lab`.
2. Paste the contents of `docs/PM_INTERVIEW_REVIEW_LAB_PROJECT_INSTRUCTIONS.md` into the Project instructions/custom instructions area.
3. Optionally upload `templates/session-tracker/review_lab_prompt.md` as a Project source, or keep it available to paste at review time.
4. Do not add the main PM Interview Helper runtime files.
5. Do not add tracker session data as permanent Project source. Session files should be attached/pasted per review session.

## Operating rule

This Project reviews interview-session evidence and produces candidate improvements. It does not directly update the main PM Interview Helper source after every session. Source updates happen later only when there is a repeated pattern or urgent truth-safety issue.
